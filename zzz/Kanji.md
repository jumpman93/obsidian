 # [[Kanji]]
 - learn the [[radicals]] rather than the stroke, radicals reduce the need to remember many strokes for complicated [[Kanji]]
 - use [[mnemonics]] stories to help memorize [[Kanji]]. Recognizing the [[radicals]] in the Kanji and building a story with them will make it easier to remember the Kanji.
